# Image Authenticity Verification Using Digital Image Forensics Techniques

A desktop-based digital image forensics tool for checking image authenticity using Error Level Analysis (ELA), EXIF metadata analysis, forensic indicators, and verdict scoring.

## Features

- Image upload and validation for JPG, JPEG, and PNG files
- Error Level Analysis (ELA)
- EXIF metadata extraction
- Forensic warning indicators
- Rule-based manipulation score
- Final authenticity verdict
- PDF forensic report export
- Beginner tutorial and glossary

## Technologies Used

- Python
- Tkinter
- Pillow
- OpenCV
- NumPy
- ReportLab
- pypdf
